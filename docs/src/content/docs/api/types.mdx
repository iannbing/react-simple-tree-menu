All types are exported from `react-simple-tree-menu`.

```ts
import type {
  TreeMenuProps,
  TreeMenuHandle,
  TreeMenuItem,
  TreeMenuChildren,
  TreeMenuClassNames,
  TreeMenuLabels,
  TreeNode,
  TreeNodeObject,
  TreeNodeInArray,
  LocaleFunction,
  MatchSearchFunction,
  Item,
} from 'react-simple-tree-menu';
```

## `TreeNode`

Shared shape for a node in either data format. Label is optional — nodes
without a label render as `"unknown"`.

```ts
interface TreeNode {
  label?: string;
  nodes?: TreeNodeObject | TreeNodeInArray[];
  // ...any custom user properties
}
```

## `TreeNodeObject`

Object data format. Keys become node `key`s.

```ts
interface TreeNodeObject {
  [key: string]: TreeNode;
}
```

## `TreeNodeInArray`

Array data format. Every node must supply a unique `key`.

```ts
interface TreeNodeInArray extends TreeNode {
  key: string;
}
```
