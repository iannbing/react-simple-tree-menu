## Highlights

<CardGrid stagger>
  <Card title="Zero runtime deps" icon="rocket">
    Ships a ~3KB bundle. React is the only peer.
  </Card>
  <Card title="Accessible" icon="approve-check">
    WAI-ARIA tree pattern, roving tabindex, full keyboard navigation out of the box.
  </Card>
  <Card title="Render-props" icon="setting">
    Use the default UI, or pass a function child and render whatever you want.
  </Card>
  <Card title="Headless-ready" icon="puzzle">
    Tailwind v4 theme variables detected automatically. Or pass `classNames` and skip our CSS.
  </Card>
  <Card title="Search with debounce" icon="magnifier">
    Built-in search filters the tree and auto-expands matching ancestors.
  </Card>
  <Card title="Controlled or uncontrolled" icon="forward-slash">
    Let the component own state, or drive `openNodes` / `activeKey` / `focusKey` yourself.
  </Card>
</CardGrid>
